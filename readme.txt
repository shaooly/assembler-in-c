In this zip folder there are all the files, the makefile and all the runnings of the assembler

working1 - an example run of a working assembly file
working2 - another example run of a working assembly file
runningtogether - running working1 and working2 together

preasm_error - like the name suggests, error in the preasm stage
firstpassage_error - like the name suggests, error in the firstpassage stage
secondpassage_error - like the name suggests, error in the secondpassage stage
